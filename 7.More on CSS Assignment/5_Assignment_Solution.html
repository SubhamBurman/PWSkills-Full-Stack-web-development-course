Google Fonts is a computer font and web font service owned by Google. This includes free and open source font families, an interactive web directory for browsing the library, and APIs for using the fonts via CSS and Android


Google Fonts is a collection of free, open-source fonts that can be used on websites. Google Fonts launched in 2010 and is the internet's largest selection of fonts. All Google Fonts are free for commercial and personal use. Google Fonts are hosted by Google and can be accessed through Google's content delivery network (CDN). Fonts are pulled into your website through the CDN and will load automatically once installed. You can also download all fonts for installation on local systems.
Google Fonts includes:

  • Interactive application programming interfaces
  • A robust catalog of open-source fonts and icons
  • Free and open source font families
  • Some examples of Google Fonts include:
    • Roboto
    • Sometype Mono
    • Playpen Sans
    • Noto Sans Osmanya
    • Open Sans
You can apply a font by adding a stylesheet link to your web page.
